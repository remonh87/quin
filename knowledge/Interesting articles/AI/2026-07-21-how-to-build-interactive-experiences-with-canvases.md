### Core News Story

- **New Canvas Extensions in GitHub Copilot**: Introduces shared, interactive workspaces where developers and AI agents collaborate in real time, enabling direct interaction with information for tasks like issue triage, visualization, and workflow management.

- **Key Use Cases Demonstrated**:
  - **Issue Triage Helper**: Card-based interface for swiping through GitHub issues to accept/reject.
  - **Interactive Codebase Diagram**: Dynamic visualization of project structure and component relationships.
  - **Sessions Worktree View**: Visual tool to manage and clean up Git worktrees and Copilot sessions.
  - **Agent Prompt Coach**: Analyzes past prompts to suggest improvements for better AI interactions.
  - **Knowledge Finder**: Searches across Slack, Teams, email, and docs to find experts on specific files/topics.

- **Implementation**: Canvases are created via `/create-canvas` in the GitHub Copilot app, evolving through iterative prompts and direct interactions.
